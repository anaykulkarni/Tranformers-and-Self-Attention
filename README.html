Instructions to Run the Code

The code can be executed by running main.py. Each part of the assignment can be accessed through arguments passed to the Python shell. Ensure all supplementary Python modules (.py files) are in the same directory as main.py.

Running Each Part of the Assignment:

  • Part 1: python main.py --model part1
  • Part 2: python main.py --model part2
  • Part 3: python main.py --model part3

Alternatively, you can use run.ipynb to execute each part in a Jupyter notebook environment.

Code Structure

  • Transformer Classes: Defined in transformers.py.
  • Utilities: Located in Utilities.py, which has been modified to improve plot readability.

Output and Reports

  • Images and Plots: Execution images and plots are saved in the /images folder.
  • Reports: The report is available in both PDF and HTML formats under the /images folder. The HTML version includes output from the executions.

Hyperparameters

Hyperparameters are globally defined at the top of main.py, corresponding to the base settings in the PA2 PDF. For Part 3, specific hyperparameters are updated directly in the main() function.

Note: The code for programmatically printing and calculating the model’s total learned parameters has been commented out. To use this feature, uncomment the relevant blocks in each section. The torchinfo library is used for generating the model summary.
